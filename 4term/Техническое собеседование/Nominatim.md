# Nominatim
В качестве примера рассмотрим сервис, который помогает работать с геоданными, его можно интегрировать по API практически везде, где требуется работа с координатами и адресами.

Мы условно бедные разработчики, у нас нет деняк на Яндекс API, а они на минуточку требуют от 120 000 Р в год при минимальном колличестве запросов, а мы только открылись и у нас только кофе с печеньками в безлимитном доступе(только если кто-то не начнет утаскивать домой)
